# AI Product Engineer Roadmap

A self-paced, mastery-based curriculum for becoming someone who deeply understands AI AND can ship complete products.

An **AI Product Engineer** owns the entire lifecycle of AI-powered products—from ideation to production at scale—combining product thinking, engineering execution, and business acumen.

---

## Phases

### Phase 1: Foundations (Units 1-13)
*Build mathematical intuition, classical ML mastery, Git proficiency, and ethical foundations*

| Unit | Title | Key Topics |
|------|-------|------------|
| 1 | Linear Algebra & Vectors | Vector operations, linear combinations, matrix transformations |
| 2 | Matrix Theory & Determinants | Determinants, inverses, linear systems, LU decomposition |
| 3 | Vector Spaces & Eigenvalues | Subspaces, basis, eigendecomposition, PCA foundations |
| 4 | Calculus for ML | Derivatives, gradients, chain rule |
| 5 | Multivariable Calculus | Partial derivatives, gradients, Jacobians, Hessians |
| 6 | Probability Foundations | Distributions, Bayes' theorem, conditional probability |
| 7 | Statistics for ML | MLE, hypothesis testing, confidence intervals |
| 8 | Python for Data Science | NumPy, Pandas, Matplotlib, Scikit-learn |
| 9 | Classical ML Algorithms | Linear/Logistic Regression, Decision Trees, SVMs, K-means, PCA from scratch |
| 10 | Data Structures & Algorithms | Essential DSA for ML implementation |
| 11 | Computational Complexity | P vs NP, algorithm analysis, intractability |
| 12 | AI & ML Ethics | Bias, fairness, data ethics, transparency, case studies |
| 13 | Git Fundamentals | Core commands, branching basics, mental model, remotes |
